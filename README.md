# java-chat-application
A Java-based multi-client chat application with GUI, private chat, and file transfer.

This application follows a client-server architecture where multiple clients connect to a central server and communicate in real time. It is developed using Java socket programming, multithreading, object serialization, and a Swing-based graphical user interface.

## Features
- Multi-client real-time chat
- Private messaging using @username
- File sharing between clients
- GUI-based client using Java Swing
- WHOISIN command to list connected users
- Chat logging to file
- Multithreaded server to handle multiple clients

## Technologies Used
- Java
- Socket Programming
- Multithreading
- Java Swing
- Object Serialization
- File Handling
